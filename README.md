# Crystal Pricing Engine

## Overview
This skill calculates structured pricing estimates for crystals using deterministic rules. It converts product details such as weight, quality, and cut into consistent wholesale and retail price ranges.

## Why I built this
Pricing crystals can be inconsistent and subjective. I wanted a repeatable system that uses defined rules to generate consistent pricing outputs for my crystal business.

## How it works
The skill:
- Takes structured inputs (stone, weight, quality, cut)
- Applies base price-per-gram values
- Uses multipliers for quality and cut
- Calculates wholesale and retail ranges

The Python script performs the deterministic calculations, ensuring consistent results that cannot be reliably generated by prompting alone.

## How to use
Provide inputs such as:

"Labradorite, 120 grams, A quality, polished"

The agent will:
1. Extract inputs
2. Run the script
3. Return pricing estimates

## Script details
The script:
- Stores base pricing data
- Applies multipliers
- Calculates wholesale and retail ranges

## What worked well
- Clear separation between AI reasoning and deterministic logic
- Reusable pricing structure
- Easy to extend with new crystal types

## Limitations
- Does not use real-time market data
- Simplified pricing model
- Not a formal appraisal tool
